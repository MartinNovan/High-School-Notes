## Elektro
###  Základní pojmy
1. **Elektrické napětí**  
	1. značka → **U**  
	2. jednotka → **V (volt)**  
	3. čím se měří → **voltmetrem**  
2. **Elektrický proud**  
	1. značka → **I**  
	2. jednotka → **A (ampér)**  
	3. čím se měří → **ampérmetrem**  
3. **Elektrický odpor**  
	1. značka → **R**  
	2. jednotka → **Ω (ohm)**  
	3. čím se měří → **ohmmetrem**  
4. **Ohmův zákon**  
	1. o čem hovoří → Vyjadřuje vztah mezi **napětím, proudem a odporem**.  
	   - Čím větší odpor, tím **menší proud** při stejném napětí.  
	2. vzorce:  
	   - `I = U / R`  
	   - `U = R * I`  
	   - `R = U / I`
5. **Příkon**  
	1. značka → **P**  
	2. jednotka → **W (watt)**  
	3. čím se měří → **wattmetrem**  
	4. význam → Udává, **kolik energie zařízení spotřebuje**.
6. **Výkon**  
	1. značka → **P**  
	2. jednotka → **W (watt)**  
	3. význam → Udává, **kolik energie zařízení dodá nebo vykoná**.
7. **Účinnost**  
	1. značka → **η (éta)**  
	2. vzorec → `η = Pv / Pp`  
	3. jednotka → **% nebo bezrozměrná hodnota (0–1)**  
	4. význam → Udává, **jak účinně zařízení přeměňuje přijatou energii na užitečnou**.
8. **Matematický vztah mezi příkonem, účinností a výkonem:**  
	- `Pv = Pp * η`  
	- `Pp = Pv / η`
9. **Teplo**  
	1. značka → **Q**  
	2. jednotka → **J (joule)**  
	3. význam → Vyjadřuje **množství energie přeměněné na teplo** (např. při průchodu proudu vodičem).  
10. **Přístroj pro měření odebrané elektrické energie:**  
	- **Elektroměr**
11. **Odebraná elektrická energie**  
	1. jednotka → **Wh nebo kWh (kilowatthodina)**  
	2. význam → Udává, **kolik elektrické energie bylo spotřebováno**.
12. **Označení „DC“ a „AC“:**  
	- **DC (Direct Current)** → stejnosměrný proud  
	- **AC (Alternating Current)** → střídavý proud  
13. **Která veličina způsobuje zahřívání vodiče?**  
	- **Elektrický proud (I)** — při průchodu vodičem vzniká **tepelný ztrátový výkon** `P = I² * R`.
14. **Pojistka / Jistič**  
	- a. schématická značka:  
	  - pojistka → `—[ ]—`  
	  - jistič → `—[⏚]—` *(zjednodušeně, liší se podle norem)*  
	- b. chrání před: **nadproudem** a **zkratem**  
	- c. připojuje se na: **fázový vodič (L)**  
15. **Co znamená „1 kWh“ vlastními slovy?**  
	- Množství energie, které zařízení s příkonem **1 kW** spotřebuje nebo dodá za **1 hodinu**.
16. **Živá část elektrického zařízení**  
	- část, která je **za normálního provozu pod napětím**, např. **fázový vodič, svorka, kontakt**.
17. **Neživá část elektrického zařízení**  
	- část, která **není pod napětím**, ale **může se dostat pod napětí při poruše**, např. **kovový kryt spotřebiče**.
